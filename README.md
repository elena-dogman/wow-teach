# WOWTeach 🎓✨

WOWTeach — это современная **LMS-платформа** для преподавателей. Здесь вы можете создавать, публиковать и управлять курсами, а также взаимодействовать с учениками и отслеживать их успехи.

---

## 🛠️ Стек технологий

- 🌐 **Next.js 14**
- ⚛️ **React**
- 🎨 **Tailwind CSS**
- 🗄️ **Prisma + MySQL**
